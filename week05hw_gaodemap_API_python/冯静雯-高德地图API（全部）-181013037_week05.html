简单的应用 ——智能会议记录APP¶

产品需求¶

  • 在日常工作中会有各种会议，为的是整合想法，了解进度，总结改进。会议中的内容非常重要，有时信息庞大且重要，参会人员数量不确定，人工记录难免受限于效率与能力，很可能出现漏重点甚至错误的情况。
  • 为保证信息完整且正确，我产生自动会议记录APP的想法。 #### 产品说明
  • 预先录入人声，身份确定，可进行部门分组，在记录中分角色或组别，进行边识别边分组。
  • 把语音转为文本记录，并进行文本分析，把关键点抽出，有详略分析版本。
  • 自动把感叹词与口语化语言去掉或转化为书面语言。
  • 可按角色或组别单独提取出单独会议记录。 #### 应用API
  • 语音转文本
  • 语言理解（在应用、机器人和 IoT 设备中实现对自然语言的理解。）
  • 文本分析（检测情绪、关键短语和命名实体。）
  • 说话人识别

ML、AI¶

1.文字识别：地理、逆地理编码、路径规划、行政区域查询、搜索POI、IP定位、天气查询、输入提示、交通态势
2.推荐算法：路径规划、输入提示
3.计算机视觉：路径规划、静态地图、交通态势、轨迹纠偏
